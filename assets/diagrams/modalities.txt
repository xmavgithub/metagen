Modality → Objective Mapping
============================

┌───────────────────┬────────────────────────────────────────────────────────┐
│     Modality      │              Recommended Objectives                    │
├───────────────────┼────────────────────────────────────────────────────────┤
│ text → text       │ autoregressive, masked                                 │
│ image → image     │ diffusion (generation), classification (ViT)          │
│ text → image      │ diffusion                                              │
│ audio → audio     │ autoregressive                                         │
│ text → video      │ diffusion                                              │
│ text+image → both │ contrastive                                            │
└───────────────────┴────────────────────────────────────────────────────────┘
