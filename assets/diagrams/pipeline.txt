Key Properties
==============

• Deterministic: Same spec + seed = identical outputs
• Multi-modal: Text, Image, Audio, Video, 3D, Time Series, Graph
• Task-aware: Classification, detection, segmentation, RL
• Extensible: Modality handlers, task handlers, architecture families
• Complete: Code, docs, benchmarks, paper in one run
